Maintenance Results

This is the tab where most of the work is actually performed.
All the selected/entered/imported servers are listed here, sorted and grouped by domain name.

The selected maintenance process then starts, and each server in turn is actioned.

If the selected action (start/update/stop) is successful, three green ticks () should be shown next to the server name.

If, for any reason an issue occurs, the icons shows could either be a yellow triangle () or a red cross (), depending on the issue at hand. A note in the auditing log will explain the exact issue.

Generally,

  • If a green tick appears, the maintenance action was successful,
  • If a yellow triangle is shown along side green ticks for a server, then that particular monitoring item doesn't exist, or has issues,
  • If a red cross appears, it is usually on its own, and could indicate a missing server, in which case, check you have entered it correctly.
