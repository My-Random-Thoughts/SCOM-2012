Server List Options

Once you have a list of servers you want to manage, the following options apply to the entire server list

  • Duration Or Specific Time
  • Category
  • Comment
  • WhatIf.?
  • Schedule Task
These are discussed below...

Duration Or Specific Time

This option will set the length of the maintenance window.
You can either set a duration of the maintenance window from 10 minutes to 86400 minutes (60 days), or you can select a specific date and time in the future.
Make sure you set enough time to perform the work, more is better in this case.

Category

This will set the SCOM Maintenance mode reason status for the server(s)
If the maintenance mode is a planned one, tick the box, or untick for an unplanned maintenance window.
From the drop down list, select the appropriate option

For more information about maintenance mode categories, see the next section on the left.

Comment

This will set the comment for the maintenance mode reason.
There are two reasons that SCOM uses, one is internal to the application, the other you set here. The internal one is set to Planned Operating System Reconfiguration

WhatIf.?

Just like in Microsoft's PowerShell, this option allows you to run though the entire process, including creating the auditing log, but without making any actual changes.

Schedule Task

This will allow you to schedule future maintenance mode operations.
See the main help note for more information.
