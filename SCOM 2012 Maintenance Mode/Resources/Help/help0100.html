Server Entry List

This is the first screen of this application, and the one where you create the server list you want to action against.

From here you can start typing the FQDN of the servers you want to manage.
Other options include importing an existing list of servers, along with optional control codes, or
Searching SCOM for all servers, filtering, and selecting the ones you want.

Select from one of the sub-sections on the left for more information.
