Application Defaults

This is where the default settings for the application are set.

These options allow you to set the default values for the application so that you don't have to set them every time you launch the application.

Currently, only the maintenance window time-out and comment reason can be set. More options may appear later.

Disallowed

This is a list of servers that will not show up in the search or validate functions, and therefore are not selectable to enter maintenance mode. They are also checked during maintenance activity so that automatic lists do not contain these servers.
These server names must match exactly how they appear in the SCOM management console.
