Search SCOM

If you are unsure of the exact DNS Display Name of the servers you want to add, you can click the Search SCOM button.

A new tab will appear and the application will connect to the specified SCOM Management server and show a list of currently managed servers. This list is then sorted and grouped by domain name.
Any servers that are currently in maintenance mode are shown with a little icon () next to their name .

Once you have found one or more of the servers you want to use, simply tick the check box next to the name, and if required, keep searching for more servers.
All previously ticked servers will be remembered. When you are done, click the Add button to add the selected servers to the server list.

Additional options are available on the right-click menu...

  •   Select All
  •   Select None
  •   Invert Selection
