Search Text

By entering text into the Search Text field, the list can be filtered to only show the servers you need.

This search is not case-sensitive, and will search for partial matches too. Wild-card characters ('*', '?')are not supported,
however multi-part searches can be achieved by using space as a separator.

Special searches can also be performed as part of a text search
By entering and exclamation mark ( ! ) followed by a specific letter shown below, the search results can be filtered even further...

  • !e      servers in an error state
  • !w      servers in a warning state
  • !o      servers marked as OK
  • !m      servers in maintenance mode
  • !n      servers not being monitored

  • !c      currently selected (checked) servers

Examples

Entering the search text CTX will search all the managed servers with this text.

Using two or more search terms can help reduce the number of servers down to a few specific ones when dealing with a lot of servers.
For example, entering LOC DOM 01 will show all servers with ALL of the specified terms, even if you are unsure of the exact server names.

Entering DOM !e will show all servers currently in an error state, with DOM in part of their name.
