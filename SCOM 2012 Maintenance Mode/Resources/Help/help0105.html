Scheduled Task

A scheduled task can be created to run this application along with a pre-existing server list
Note: To make sure the application runs smoothly,

  • the server list must include the control code #auto-start,
  • the server list should be located in a non-user specific location (away from user profiles),
For more information on control codes, select the topic on the left.

When you click the Schedule Task button, a question dialog will ask if you have a pre-existing server list. Click Yes to continue.

From the window that appears enter the following details...

  • Task Name and Description
    The description can be automatically generated from the server list if required,
  • Username and Password
    The account that the scheduled task is to be run against,
  • Start Time
    The time the task is to start,
  • Server List
    The pre-existing server list with control codes that is to be used.

Once completed, press Save to create the scheduled task. A message will appear to inform you if the task creation was successful or not.
