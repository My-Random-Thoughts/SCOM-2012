Import

There are two ways to import an existing list of servers into this application...

  1. Click the Import button, and
  2. Drag and drop the file over the application.

By clicking the button, a window will appear asking which file to import. It must be a plain text file with one server per line.
The imported file will have basic error checking done on it, but nothing more.

Note: Server names must match the DNS Display name entry in SCOM, otherwise they will not be found.

If the imported file contains specific control codes, they will be used by the application to override the default settings.
Either equals (=) or colon (:) can be used

See the section Control Codes for a full list
